---
description: Stage all changes and commit them to the current branch
---

1. Stage all changes
```bash
git add .
```

2. Review the staged changes to understand what was modified.
// turbo
```bash
git diff --cached
```

3. Commit changes using a comprehensive and detailed commit message that accurately describes the changes you just made and the diff you reviewed.
// turbo
```bash
git commit -m "<YOUR_COMPREHENSIVE_AND_DETAILED_COMMIT_MESSAGE>"
```
